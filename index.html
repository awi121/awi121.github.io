PUNO

This project develops a hybrid physics-informed neural network operator called PUNO for solving partial differential equations. PUNO first learns an operator over a family of PDEs using both data and physics constraints, then fine-tunes the operator for a specific PDE instance using additional physics-based losses.

Attention-based Fusion Network for 3D Semantic Segmentation

This project develops an attention-based fusion network for 3D semantic segmentation of LiDAR point clouds by projecting to range images, extracting features with self-attention, fusing features with cross-attention, and segmenting using a convolutional network before projecting back to the point cloud.

Multimodal multihop question answering

The project explores multihop multimodal QA on the WebQA dataset through ideas such as using universal embeddings for retrieval and generation, decomposing complex questions into simpler sub-questions, and altering the training objective to exploit emergence properties of QA tasks.
